# Chompy O'Croc

Run and jump around platforms, collecting stars and avoiding bombs!

## Technologies

The game was made with [Phaser](https://phaser.io/), [Vite](https://vitejs.dev/), and [TypeScript](https://www.typescriptlang.org/).
